# Protocol Site

Frontend-сайт для игры **Protocol**. Проект разрабатывается как официальный веб-сайт игры: главная страница, навигация, информационные разделы, новости, авторизация, регистрация и профиль игрока.

Сайт демонстрирует работу с компонентной архитектурой React, адаптивной версткой, кастомной навигацией, UI-состояниями и подготовкой frontend-слоя к будущей интеграции с backend/API.

## Технологии

- React
- Vite
- JavaScript
- Tailwind CSS
- CSS
- Git

## Что реализовано

- Адаптивная верстка для desktop и mobile.
- Компонентная архитектура интерфейса.
- Навигация между страницами через локальный router.
- Главная страница в темном sci-fi стиле.
- Header, Footer, hero-блок, карточки особенностей и блок новостей.
- Интерактивные элементы: hover/focus состояния, кнопки, меню, анимация персонажа.
- Подготовлены страницы входа, регистрации и профиля пользователя.
- Mock-слой авторизации на `localStorage` для проверки пользовательского сценария.
- Описана структура данных для будущей backend-интеграции.

## Установка и запуск

```bash
git clone https://github.com/Superdeazi2/protocol-site.git
cd protocol-site
npm install
npm run dev
```

## Основные разделы проекта

- **Главная страница** — промо-экран проекта Protocol с визуальным оформлением, CTA, новостями и карточками особенностей.
- **Навигация** — Header с активными пунктами меню и адаптивным мобильным меню.
- **Авторизация** — страница входа с валидацией формы и mock-проверкой пользователя.
- **Регистрация** — создание локального пользователя для демонстрации UX-сценария.
- **Профиль пользователя** — компактная игровая карточка с данными игрока и выходом из аккаунта.
- **Информационные разделы** — подготовлены маршруты для новостей, обновлений, информации об игре и сообщества.

## Особенности UI

- Темный sci-fi стиль с фиолетовыми акцентами.
- Адаптивные сетки и компактные карточки.
- Тонкие рамки, glow-эффекты и игровые визуальные детали.
- Hover/focus состояния для интерактивных элементов.
- Локальные шрифты и читаемая типографика.
- Покадровая анимация персонажа на главной странице.
