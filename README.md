# Protocol

Frontend-проект игрового веб-сайта, разработанный на React и Vite.

Проект создается как официальный сайт игры Protocol и используется для публикации новостей, обновлений, описания игровых механик и взаимодействия с сообществом игроков.

## Что реализовано

* Адаптивная верстка для мобильных устройств и ПК
* Навигация между страницами
* Компонентная архитектура на React
* Современный UI в стиле Sci-Fi
* Анимации интерфейса
* Подготовка структуры для авторизации пользователей
* Подготовка к интеграции с backend

## Технологии

* React
* Vite
* JavaScript
* CSS
* Git

## Основные задачи проекта

* Разработка интерфейсов по собственным макетам из Figma
* Создание переиспользуемых компонентов
* Проработка UX и адаптивности
* Оптимизация структуры проекта для дальнейшего масштабирования

## Запуск проекта

```bash
Склонировать репозиторий по SSH в нужную папку, запуск:

git clone git@github.com:Superdeazi2/protocol-site.git
npm install
npm run dev
http://localhost:5173
```
