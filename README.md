# Protocol

Сайт для моей собственной игры **Protocol**.

Дизайн сайта был разработан мной в Figma, после чего я начал поэтапно реализовывать его на React.

## О проекте

Protocol — это 2D roguelike игра, которую я разрабатываю самостоятельно. На данный момент у игры уже есть рабочий MVP: основные механики реализованы и в нее можно играть. Сейчас проект продолжает активно развиваться и наполняться новым контентом.

Этот сайт создается как официальный сайт игры. Здесь будут размещаться новости, обновления, информация об игровом мире и другие материалы, связанные с проектом.

## Технологии

* React
* Vite
* JavaScript
* CSS

## Статус проекта

Сайт и игра находятся в активной разработке. Новые страницы, функции и визуальные элементы постепенно добавляются по мере развития проекта.
